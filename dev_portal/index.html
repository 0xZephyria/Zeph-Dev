◉ MetaMask Wallet Connection

1. Connect MetaMask to Zephyria Node
  • Open MetaMask, click the network dropdown and select "Add network".
  • Click "Add a network manually".
  • Enter the following details:
    • Network Name: Zephyria Devnet
    • RPC URL: http://127.0.0.1:8545
    • Chain ID: 91919191
    • Currency Symbol: ZEPH
2. Import Pre-funded Devnet Account
  • In MetaMask, select "Import Account" under accounts.
  • Paste the devnet private key:
    • ac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80
  • This imports the account 0xf39Fd6e51... loaded with 100,000 ZEPH.

◉ Remix IDE Contract Deployment

Deploy from Remix
  • Open remix.ethereum.org.
  • Navigate to the Deploy & Run Transactions tab.
  • Under **Environment**, select "Injected Provider - MetaMask".
  • MetaMask will prompt to connect to the Zephyria Devnet.
  • Now you can compile and deploy standard Solidity contracts directly via Remix!
